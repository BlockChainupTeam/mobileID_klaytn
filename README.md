# 실행방법
현재 리포 받은 후 
> npm install <br>
> truffle-config.js에서 PrivateKey 변경 <br>
> npm run dev //실행

- node.js: 10.15.3
- npm: 6.4.1
- caver-js: v1.5.2-rc.1
- truffle: 4.1.15
- solidity: 0.4.25
- using webpack
